# Logo Light Agent Command Center

Initial GitHub-hosted dashboard pilot.

## Current capabilities
- Executive overview
- LinkedIn and YouTube connection status
- Social calendar prototype
- Listings/search status
- Visible last-updated timestamp
- Automatic GitHub Pages deployment

## Important
This is still a static pilot. It does not yet run autonomous agents or store credentials.

## GitHub Pages setup
1. Create a public repository named `logo-light-command-center`.
2. Upload all files from this package, including the `.github` folder.
3. Open repository **Settings → Pages**.
4. Under **Build and deployment**, select **GitHub Actions**.
5. The workflow will publish the dashboard.
