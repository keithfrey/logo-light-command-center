FIRST LAUNCH

1. Copy all files from this package into:
   C:\logo-light-command-center

2. In File Explorer, open that folder.

3. Click the address bar, type:
   powershell
   and press Enter.

4. Run:
   docker compose up --build -d

5. Wait 1–3 minutes.

6. Open:
   http://localhost:3000

7. In GitHub Desktop:
   Summary: Add first live command center stack
   Commit to main
   Push origin
